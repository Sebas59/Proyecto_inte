  • FastAPI: Para un backend rápido y robusto.

  • Jinja2: Para la renderización dinámica de templates HTML.
  • Bootstrap: Para un diseño frontend moderno y responsivo.
  • Control de Versiones (Git): Para una gestión organizada del código.

Mi objetivo es construir soluciones prácticas y fáciles de usar que resuelvan problemas reales. ¡Espero que encuentres esta aplicación útil!
